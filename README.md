#  python-stream

### 说明
这是一个数据结构化框架, 可用作数据清洗, 数据结构化预处理, 数据管道等应用场景


### 安装
-----------
```shell
pip install git+https://github.com/sandabuliu/python-stream.git
```
or

```shell
git clone https://github.com/sandabuliu/python-stream.git
cd python-agent
python setup.py install
```
